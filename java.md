# Java

## Java 언어 특징
 * 소스 코드(java 언어) 자체는 플랫폼에 독립적
 * javac 에 의해 .class 로 만들어 짐.
 * JVM 자체 연관이 타이트하지 않음. (JVM 은 .class 만 다룸)

## JVM
 * .class 파일을 OS 에 맞는 machine code 로 변환 (인터프리터 & JIT 컴파일러)

## JRE (Java Runtime Environment)
* JVM + 핵심 라이브러리
* 개발 관련 x. 실행 관련 o

## JDK (Java Development Kit)
* JRE + 개발 툴(javac)
